# E-commerce-webshop-selenium-java

## Overview
This project demonstrates functional and UI test automation for an e-commerce web application ([Swag Labs](https://www.saucedemo.com/)).

The test suite is implemented using Selenium WebDriver with JUnit Jupiter and includes automated verification of authentication, cart functionality, navigation, and key UI elements.

### Test Coverage
The automated tests cover the following areas:

- Authentication scenarios (valid and invalid login)
- Access control validation
- Product cart functionality (add/remove products)
- Cart state persistence during navigation
- Sidebar navigation and logout
- UI verification of selected components
- Parameterized tests using external test data
- Test reporting with Allure

## Setup Instructions
### Prerequisites
* [![Java][JavaSmall]][java-url]
* [![Maven][MavenSmall]][maven.url]

```bash
java -version
mvn -version
```


## Running the Tests
```bash
mvn clean test
```

### Generate and open Allure report
```bash
allure serve target/allure-results
```
